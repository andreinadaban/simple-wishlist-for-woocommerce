A simple extension for WooCommerce that allows your customers to add products to a wishlist.

== Description ==
This is a simple extension for WooCommerce that allows your customers to add their favorite products to a private wishlist. This AJAX powered plugin provides a better user experience by allowing products to be added to the wishlist almost instantly*, without reloading the page.

* Assuming the website runs on a decent server that meets the minimum system requirements for WordPress.
